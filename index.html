이 포트폴리오에서는 웹 퍼블리셔를 꿈꾸며 준비한, 저의 노력의 결과물인 작업을 관람하실 수 있습니다. 화려한 디자인보다도, 최적의 사용자 경험을 제공하는 것을 목표로 하는 웹 퍼블리셔입니다!

  • 콘텐츠가 명확하고 일관성 있게 구성된 페이지를 만듭니다. 간단한 사용자 경로, 명료한 레이아웃 및 직관적인 내비게이션은 모든 사용자에게 도움이 됩니다. 따라서 다양한 사용자 그룹을 고려하여 웹 페이지를 디자인하고 개발함으로써 웹을 보다 포괄적으로 사용할 수 있도록 합니다.
  • 코드를 작성할 때, 단순히 빠르고 올바르게 동작하는 것으로만 만족하지 않고, 유지 보수와 여러 가능성에 대해 고려하며 개발을 진행합니다.
  • Vanilla JavaScript · jQuery 플러그인을 사용하여 웹 사이트의 성능과 사용자 경험을 높일 수 있습니다.
  • 디자인에 대한 기본적인 이해와 감각을 가지고 있습니다. 이미지를 특정한 상황에 맞게 편집 · 디자인하여 배치할 수 있습니다.
