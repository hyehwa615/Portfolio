이 포트폴리오에서는 웹 퍼블리셔를 꿈꾸며 준비한, 저의 노력의 결과물인 작업을 관람하실 수 있습니다. 화려한 디자인보다도, 최적의 사용자 경험을 제공하는 것을 목표로 하는 웹 퍼블리셔가 되겠습니다!

  • 웹 접근성과 웹 표준을 고려한 웹 페이지를 만들고, 웹 페이지를 만들거나 수정하는 과정에서 발생할 수 있는 문제를 해결할 수 있습니다.
  • 유지 보수가 쉬운 코드를 짤 수 있도록 더 좋은 방법에 대해 끊임없이 생각하며, 여러 가능성에 대해 고려하며 개발을 진행합니다.
  • jQuery 플러그인을 사용하여 동적인 사이트를 제작할 수 있고, 자바스크립트 기초를 공부하며 더 발전하고자 노력 중입니다.
  • 디자인에 대한 기본적인 이해와 감각을 가지고 있습니다. 이미지를 특정한 상황에 맞게 편집 · 디자인하여 배치할 수 있습니다.
